# Java Email Bridge Server & Client Emulator

A lightweight Spring Boot application that simplifies sending emails through multiple vendors (Gmail, Walla, Yahoo) by acting as a centralized email bridge.  
Built with Docker, Kubernetes, and RabbitMQ for scalable, multithreaded email processing.  
This system enables seamless email routing without logging into each vendor separately, while logging email metadata for audit purposes.  
Perfect for organizations managing multiple email accounts across different providers.
